# JavaScript Sample Website

## Overview
This is a simple JavaScript-based website designed to demonstrate core JavaScript concepts using a clean blue and white UI.

## Pages / Sections
- Home (Hero Section)
- About
- Task Section
- Dashboard

## Technologies Used
- HTML
- CSS
- JavaScript

## JavaScript Features
- Alert messages
- DOM manipulation
- Event handling
- User input validation
- Simple calculations
- Date and time display
